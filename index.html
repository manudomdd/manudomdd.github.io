Desarrollador con pasión por el backend

Soy Manu, un desarrollador backend apasionado por construir sistemas que funcionan bien por dentro. Mi día a día gira en torno a diseñar arquitecturas limpias, APIs REST robustas y automatizar flujos con contenedores Docker.

Mi stack principal es Java con Spring Boot, aunque también domino Python con Django para proyectos donde encaja mejor. Soy meticuloso con el control de versiones en Git y firme creyente de que el código limpio no es un lujo, es una necesidad.
